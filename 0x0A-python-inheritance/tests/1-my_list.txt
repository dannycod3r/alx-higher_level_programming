# 1-my_list.txt

=========================
Usage of module 1-my_list
=========================

The module supplies the class ``MyList``, a custom define class
subclass of list. 

Class MyList defines a function ``print_sorted`` that  that prints the list,
but sorted (ascending sort). Assume that all the elements of the list will be of type int

creating a new list
::
    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()

adding content to list
::
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

print a sorted list
::
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

the sorting shouldn't modify the original list
::
    >>> print(my_list)
    [1, 4, 2, 3, 5]

empty list returns []
::
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

return that element if its only one
::
    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [5]

for duplicate elements
::
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(5)
    >>> my_list.append(2)
    >>> my_list.append(2)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 1, 2, 2, 3, 4, 5, 5]
